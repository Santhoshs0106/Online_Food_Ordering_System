## ğŸ” About the Project

This is a **frontend-only online food ordering web app** with a simple and intuitive interface. It is fully responsive and can be used on desktops, tablets, and mobile devices. Users can browse categories, view food details, and manage their cart in real-time.

## ğŸš€ Features

- Browse a **list of food items** with images, prices, and descriptions  
- Add or remove items from the **cart**

- Filter food items by **categories** (e.g., Pizza, Burgers, Sandwiches)  
- Responsive design for **all devices**  
- Multiple pages: Home, Menu, About, Contact
